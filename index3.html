Touch.js

Baidu Cloud Event & Gesture Library

百度云手势事件库Touch.js实现了Web App在触屏设备上的手势识别与事件管理机制
是近年来百度移动云事业部内部广泛使用的开发工具
代码开源旨在帮助更多开发者学习和开发出优秀的Web App产品

Download

Production or view source on github

Features

  • 无侵入设计, 与现有代码与框架共存
  • 原生出色的加速度感知与缓动效果, 优秀的单指旋转手势
  • 基于原生事件, 支持事件代理, 性能更好
  • 极简的API, 秒速上手, 玩转手势

Examples


//swipe example
touch.on('.target', 'swipeleft swiperight', function(ev){
    console.log("you have done", ev.type);
});
